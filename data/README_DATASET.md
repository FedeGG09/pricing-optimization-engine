# Industrial Dataset Pack

This folder contains a realistic synthetic industrial dataset generated from a public catalog structure inspired by Sullair Argentina's visible site sections.

## Included tables

- `product_master.csv`
- `account_master.csv`
- `contact_master.csv`
- `lead_funnel.csv`
- `transactions.csv`
- `audit_log.csv`
- `service_events.csv`

## Intended use

- Dynamic pricing models
- Lead scoring and prospecting
- Compliance and audit validation
- Revenue simulation
- Local RAG / vector memory for industrial operations

## Notes

The catalog is curated from public site navigation and product-family listings:
- Equipos
- Repuestos
- Postventa

The dataset is synthetic, but intentionally realistic in:
- pricing bands
- sector mix
- lead funnel stages
- service intensity
- margin behavior
- compliance rules
